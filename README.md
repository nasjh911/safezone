# SafeZone — Setup Instructions

## Before you start (one time only)
Install Node.js from https://nodejs.org — download the LTS version and install it.

## Step 1 — Add your Supabase anon key
1. Open the `.env` file in this folder with Notepad
2. Replace `PASTE_YOUR_ANON_KEY_HERE` with your anon key from:
   https://supabase.com/dashboard/project/xhanxqjkfvttzvsqtmuw/settings/api-keys/legacy
   (copy the long `eyJ...` string next to "anon public")

## Step 2 — Run the app

**Windows:** Double-click `START.bat`

**Mac/Linux:** Open Terminal in this folder and run:
```
./start.sh
```

## Step 3 — Open the app
Go to: http://localhost:5173
